# Segmentación de Vasos Retinianos con U-Net: Optimización Secuencial de Hiperparámetros

<sub> Este proyecto implementa y entrena una arquitectura U-Net para la segmentación de vasos sanguíneos en imágenes de fondo de ojo. El objetivo principal es analizar cómo cada hiperparámetro afecta al rendimiento cuando se optimiza de forma acumulativa, manteniendo los cambios que mejoran los resultados y descartando los que los empeoran.</sub>
